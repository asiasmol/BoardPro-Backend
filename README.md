# Powerboard
Kanban board web application
## Goal
Powerboard was created as a 'skill showcase' app and hopefully will help me land my first job as a Java developer.
## Technologies
Backend
* Java
* Spring Boot
* Spring Security
* JPA
* Hibernate
* JJWT
* Lombok
* PostgreSQL
* Websocket

Frontend
* Typescript
* React
* Dnd-kit
* Mui-materials

Deploy
* AWS S3
* AWS EC2
* AWS RDS
